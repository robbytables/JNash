Changing the Path of Images
(IT’S IMPORTANT):

To Change the Images:
Do not forget to put the absolute link the images of his campaign. Use the tool "Find and Replace" from his preferential editor. Change all the paths "img src="images/” (no quotes) to the location that is in your hosting server, for example, "img src="http://www.yousite.com/images/” (no quotes). Through the choice “Find and Replace” you make the change all references with a single click, easing the process of change. Change all the images, logos, links and texts as your need and enjoy professional marketing.

Modifying the Content:
Because of the way how the email clients behave with the html & CSS code, this email is built using tables for layout & inline CSS to style the elements. This is essential for the email to display correctly in most desktop & web-based emails clients.

Commented Code:

Each sections are commented of the template. You can easily find the sections you want to update.

Before Sending emails

Prior to sending any emails I strongly recommend that you to send a few test messages to test if everything displays correctly. All paths in the templates are relative - you'll need to put absolute paths ('http://...') to all images and files in order to display those correctly. All links are empty ('#') and you'll need to add specific links by yourself. Please change the title of the template.
